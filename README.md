# University rating system

**Introduction**

The project focuses on creating a university rating system through Google Spreadsheets, using the data collected via Google Forms.
The analysis is based on an Excel file, structured into separate sheets for calculations, summaries, detailed results, and reference data, ensuring an organized and systematic approach.
